# GDGN_AgenticPremierLeague2026
 Two Screens One Vibe


A Terminal Based Indian Premier League 2026 Prediction Application built using Python and Go Lang. The application provides users with real-time predictions and insights for the Indian Premier League (IPL) 2026 season, leveraging data from the SportsMonk API. Users can access match predictions, player performance analysis, and team statistics through a user-friendly terminal interface. The application is designed to enhance the IPL viewing experience by offering accurate and timely information to cricket enthusiasts.
